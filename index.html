THE ACTION ATTRIBUTE
ENGLISH VIETNAMESE WORD - 01 WORD - 02 WORD - 03
The action attribute defines the action to be performed when the form is submitted. Thuộc tính action xác định hành động sẽ được thực hiện khi biểu mẫu được gửi.
Usually, the form data is sent to a file on the server when the user clicks on the submit button. Thông thường, dữ liệu biểu mẫu được gửi đến tệp trên máy chủ khi người dùng nhấp vào nút gửi.
Tip: If the action attribute is omitted, the action is set to the current page. Mẹo: Nếu thuộc tính hành động bị bỏ qua, hành động sẽ được đặt thành trang hiện tại. omit

The target attribute specifies where to display the response that is received after submitting the form. Thuộc tính target chỉ định nơi hiển thị phản hồi nhận được sau khi gửi biểu mẫu. specify

_blank Chỗ trống The response is displayed in a new window or tab Phản hồi được hiển thị trong một cửa sổ hoặc tab mới
